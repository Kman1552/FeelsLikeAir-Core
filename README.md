# FeelsLikeAir-Core
An end-to-end cloud-native platform for real-time urban air quality monitoring and predictive health analytics. Built with Python, C++, and SQL, leveraging [AWS/GCP] for scalable data pipelines.

Phase 1 :

  - Data validation is now complete
  - it is a 3 layer validation using PyDantic V2 to ensure data integrity on readings on co-ordinates, sensors, location metadata

Tech Used here for validation : 
  - Python 3.10+
  - PyDantic v2
